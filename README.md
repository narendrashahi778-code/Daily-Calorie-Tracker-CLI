Daily Calorie Tracker CLI
Overview
The Daily Calorie Tracker CLI is a command-line application designed to help users monitor their daily calorie intake. This tool allows you to record meals, track calories, compare against a daily limit, and optionally save sessions to a file for future reference.

Features
Meal Tracking: Input multiple meals with their respective calorie values

Calorie Calculation: Automatically calculates total and average calories consumed

Limit Monitoring: Compares total intake against a user-defined daily limit

Session Saving: Option to save tracking sessions to a text file with date stamps

User-Friendly Interface: Clean command-line interface with formatted output

Installation
No installation required! This is a standalone Python script.

Prerequisites
Python 3.x installed on your system

Setup
Download the calorie_tracker.py file

Ensure Python is installed by running python --version or python3 --version in your terminal

That's it! You're ready to use the tracker.
